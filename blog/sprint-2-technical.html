CSS Position: The difference between Absolute, Relative, and Fixed positioning.

Absolute Positioning:

Absolute Positioning takes the chosen Element out of the websites flow. It will still appear in the browser, but its position will be ignored by all other Elements allowing the chosen Element to overlay any other Elements on the page. The position can be adjusted by four different values top, bottom, left, right. Note that these values push the object away from the set direction (left moves an Element to the right).

Syntax:

            
                .container {
                    position: relative;
                }

                .absolute-element {
                    position: absolute;
                    top: 10px;
                    left: 10px;
                }

Best Uses:

Absolute positioning is useful for when an element needs to overlay other sections of you webpage without breaking the positioning of other Elements. It allows you to accurately position an object on a page which means there are no spacial limitations, so long as its within its parent container. It should be used with care and only when it is impossible to position using other methods. As accurate as it is, having dozens of elements that need to be manually changed everytime something new is added will only make more work.

Relative Positioning:

Unlike Absolute positioning. The relative property keeps the element within the websites flow. But now instead of its position being based on its parent container or root, the position will be based on where its initial position was on the page (It is relative to itself!). Much like Absolute it provides the power to overlay other Elements on the page using the same top, bottom, left, right values. But wont change the initial page layout when initiated.

Syntax:

            
                .relative-element {
                    position: relative;
                    top: 85px;
                    left: 40px;
                }

Best Uses:

Relative positioning is good for when a particular Element needs to be nudged or moved to a certain position without jeapardising the positions of other Elements. Note that when another element is added, the position of the element will move with everything else. This is because the position is relative to the origin of the element, which has just been moved.

Fixed Positioning:

Fixed Positioning is the exact same as Absolute Positioning with an exception. When fixed position is applied. The element will ignore scrolling, and will stay exactly where it is on the page. (You may have noticed it when you first opened the page!).

Syntax:

            
                .fixed-element {
                    position: fixed;
                    top: 600px;
                    left: 1500px;
                }

I am a Fixed Element!

Best Uses:

Fixed positioning is great for when you want a regularly accessible feature to stay on the page. Things like chat bubbles, sidebars and help buttons are all common uses of fixed Positioning
